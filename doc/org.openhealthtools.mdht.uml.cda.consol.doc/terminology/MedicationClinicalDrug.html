Medication Clinical Drug

Value Set Medication Clinical Drug - 2.16.840.1.113762.1.4.1010.4
Code System RxNorm - 2.16.840.1.113883.6.88
Description All prescribable medication formulations represented using either a "generic" or "brand-specific" concept. This includes RxNorm codes whose Term Type is SCD (semantic clinical drug), SBD (semantic brand drug), GPCK (generic pack), BPCK (brand pack), SCDG (semantic clinical drug group), SBDG (semantic brand drug group), SCDF (semantic clinical drug form), or SBDF (semantic brand drug form). Value set intensionally defined as a GROUPING made up of: Value Set: Medication Clinical General Drug (2.16.840.1.113883.3.88.12.80.17) (RxNorm Generic Drugs); Value Set: Medication Clinical Brand-specific Drug (2.16.840.1.113762.1.4.1010.5) (RxNorm Branded Drugs).

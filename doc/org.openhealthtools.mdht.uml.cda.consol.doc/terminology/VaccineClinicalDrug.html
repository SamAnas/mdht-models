Vaccine Clinical Drug

Value Set Vaccine Clinical Drug - 2.16.840.1.113762.1.4.1010.8
Code System RxNorm - 2.16.840.1.113883.6.88
Source URL https://vsac.nlm.nih.gov/
Definition Administrable vaccine medication formulations represented using either a "generic" or "brand-specific" concept. Value set intensionally defined from RXNORM (OID: 2.16.840.1.113883.6.88), comprised of those codes whose ingredients map to NDC codes that the CDC associates with CVX codes.

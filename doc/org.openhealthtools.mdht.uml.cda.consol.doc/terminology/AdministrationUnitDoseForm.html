Administration Unit Dose Form

Value Set AdministrationUnitDoseForm - 2.16.840.1.113762.1.4.1021.30
Code System FDA RouteOfAdministration - 2.16.840.1.113883.3.26.1.1
Source VSAC
Source URL https://vsac.nlm.nih.gov/
Description Codes that are similar to a drug "form" but limited to those used as units when describing drug administration when the drug item is a physical form that is continuous and therefore not administered as an "each" of the physical form, or is not using standard measurement units (inch, ounce, gram, etc.) This set does not include unit concepts that mimic "physical form" concepts that can be counted using "each", such as tablet, bar, lozenge, packet, etc

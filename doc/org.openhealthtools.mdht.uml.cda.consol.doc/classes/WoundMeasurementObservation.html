Wound Measurement Observation

[Observation: templateId 2.16.840.1.113883.10.20.22.4.133]

This template represents the Wound Measurement Observations of wound width, depth and length.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-29928, CONF:1098-29929 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.133"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-29926)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-29927)
  4. SHALL contain at least one [1..*] id (CONF:1098-29930)
  5. SHALL contain exactly one [1..1] code, which SHALL be selected from ValueSet Wound Measurements 2.16.840.1.113883.1.11.20.2.5 DYNAMIC (CONF:1098-29931)
  6. SHALL contain exactly one [1..1] statusCode (CONF:1098-29933)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-29934)
  7. SHALL contain exactly one [1..1] effectiveTime (CONF:1098-29935)
  8. SHALL contain exactly one [1..1] value with @xsi:type="PQ" (CONF:1098-29936)

<observation xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns="urn:hl7-org:v3" classCode="OBS" moodCode="EVN">
  <templateId root="2.16.840.1.113883.10.20.22.4.133"/>
  <id root="MDHT" extension="2048659195"/>
  <code codeSystem="2.16.840.1.113883.6.1" codeSystemName="LOINC"/>
  <text>Text Value</text>
  <statusCode code="completed"/>
  <effectiveTime>
    <low value="2017"/>
    <high value="2017"/>
  </effectiveTime>
  <value xsi:type="PQ"/>
</observation>

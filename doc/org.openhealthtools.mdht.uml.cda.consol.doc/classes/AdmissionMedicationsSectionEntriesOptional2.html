Admission Medications Section (entries optional) (V3)

[Section: templateId 2.16.840.1.113883.10.20.22.2.44]

The section contains the medications taken by the patient prior to and at the time of admission to the facility.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-10098, CONF:1198-10392, CONF:1198-32560 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.44"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Hospital Admission Medications Section Entries Optional template (templateId: 2.16.840.1.113883.10.20.22.2.44)
  3. SHALL contain exactly one [1..1] code (CONF:1198-15482)/@code="42346-7" Medications on Admission (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-15483, CONF:1198-32142)
  4. If section/@nullFlavor is not present, SHOULD contain [0..*] entry such that it (CONF:1198-10102, CONF:1198-15484)
    1. Conforms to Admission Medication2 (templateId: 2.16.840.1.113883.10.20.22.4.36:2014-06-09)

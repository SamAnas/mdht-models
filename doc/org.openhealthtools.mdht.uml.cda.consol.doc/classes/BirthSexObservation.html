Birth Sex Observation

[Observation: templateId 2.16.840.1.113883.10.20.22.4.200]

This observation represents the sex of the patient at birth. It is the sex that is entered on the person's birth certificate at time of birth. This observation is not appropriate for recording patient gender (administrativeGender).

  1. SHALL contain exactly one [1..1] templateId ( CONF:3250-18232, CONF:3250-18233, CONF:3250-32949 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.200"
    2. SHALL contain exactly one [1..1] @extension="2016-06-01"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:3250-18230)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:3250-18231)
  4. SHALL contain exactly one [1..1] code (CONF:3250-18234)/@code="76689-9" Sex Assigned At Birth (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:3250-18235, CONF:3250-21163)
  5. SHALL contain exactly one [1..1] statusCode (CONF:3250-18124)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:3250-18125)
  6. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHALL be selected from ValueSet ONC Administrative Sex 2.16.840.1.113762.1.4.1 STATIC (CONF:3250-32947)

Reason For Referral Section2

[Section: templateId 1.3.6.1.4.1.19376.1.5.3.1.3.1]

This section describes the clinical reason why a provider is sending a patient to another provider for care. The reason for referral may become the reason for visit documented by the receiving provider.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-7844, CONF:1098-10468, CONF:1098-32571 ) such that it
    1. SHALL contain exactly one [1..1] @root="1.3.6.1.4.1.19376.1.5.3.1.3.1"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Reason For Referral Section template (templateId: 1.3.6.1.4.1.19376.1.5.3.1.3.1)
  3. SHALL contain exactly one [1..1] code (CONF:1098-15427)/@code="42349-1" Reason for Referral (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-15428, CONF:1098-30867)
  4. If section/@nullFlavor is not present, MAY contain [0..*] entry such that it (CONF:1098-30808, CONF:1098-30897)
    1. Conforms to Patient Referral Act (templateId: 2.16.840.1.113883.10.20.22.4.140)

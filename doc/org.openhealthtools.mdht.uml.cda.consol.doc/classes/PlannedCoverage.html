Planned Coverage

[Act: templateId 2.16.840.1.113883.10.20.22.4.129]

This template represents the insurance coverage intended to cover an act or procedure.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-31947, CONF:1098-31948 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.129"
  2. SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-31945)
  3. SHALL contain exactly one [1..1] @moodCode="INT" (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-31946)
  4. SHALL contain at least one [1..*] id (CONF:1098-31950)
  5. SHALL contain exactly one [1..1] code (CONF:1098-31951)/@code="48768-6" Payment Sources (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-31952, CONF:1098-31953)
  6. SHALL contain exactly one [1..1] statusCode (CONF:1098-31954)/@code="active" Active (CodeSystem: 2.16.840.1.113883.5.4 HL7ActCode) (CONF:1098-31955)
  7. MAY contain [0..*] author such that it (CONF:1098-32178)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  8. SHALL contain exactly one [1..1] entryRelationship such that it (CONF:1098-31967)
    1. SHALL contain exactly one [1..1] @typeCode="COMP" (CONF:1098-31968)
    2. SHALL contain exactly one [1..1] act such that it (CONF:1098-31969)
      1. SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-31970)
      2. SHALL contain exactly one [1..1] @moodCode="INT" (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-31971)
      3. SHALL contain at least one [1..*] id (CONF:1098-31972)

        These act/identifiers are unique identifiers for the policy or program providing the coverage.

      4. SHALL contain exactly one [1..1] code, which SHALL be selected from ValueSet Source of Payment Typology (PHDSC) 2.16.840.1.114222.4.11.3591 DYNAMIC (CONF:1098-31973)
      5. SHALL contain exactly one [1..1] statusCode (CONF:1098-31974)/@code="active" Active (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-31975)

Vital Signs Organizer (V3)

[Organizer: templateId 2.16.840.1.113883.10.20.22.4.26]

This template provides a mechanism for grouping vital signs (e.g., grouping systolic blood pressure and diastolic blood pressure).

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-7281, CONF:1198-10528, CONF:1198-32582 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.26"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Vital Signs Organizer template (templateId: 2.16.840.1.113883.10.20.22.4.26)
  3. SHALL contain exactly one [1..1] code (CONF:1198-32740)/@code="46680005" Vital Signs (CodeSystem: 2.16.840.1.113883.6.96 SNOMEDCT) (CONF:1198-32741, CONF:1198-32742)

    Compatibility support for C-CDA R1.1 and C-CDA 2.1: A vitals organizer conformant to both C-CDA 1.1 and C-CDA 2.1 would contain the SNOMED code (46680005) from R1.1 in the root code and a LOINC code in the translation. A vitals organizer conformant to only C-CDA 2.1 would only contain the LOINC code in the root code.

  4. SHALL contain exactly one [1..1] effectiveTime (CONF:1198-7288)

    Note: The effectiveTime is an interval that spans the effectiveTimes of the contained vital signs observations.

  5. SHOULD contain [0..*] author such that it (CONF:1198-31153)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  6. SHALL contain [1..*] component such that it (CONF:1198-7285, CONF:1198-15946)
    1. Conforms to Vital Sign Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.27:2014-06-09)

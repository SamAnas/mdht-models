Nutrition Assessment

[Observation: templateId 2.16.840.1.113883.10.20.22.4.138]

This template represents the patient's nutrition abilities and habits including intake, diet requirements or diet followed.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-30326, CONF:1098-30327 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.138"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-30324)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-30325)
  4. SHALL contain at least one [1..*] id (CONF:1098-30328)
  5. SHALL contain exactly one [1..1] code (CONF:1098-30329)/@code="75303-8" Nutrition assessment (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-32784, CONF:1098-32785)
  6. SHALL contain exactly one [1..1] statusCode (CONF:1098-30332)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-30333)
  7. SHALL contain exactly one [1..1] effectiveTime (CONF:1098-31666)
  8. SHALL contain exactly one [1..1] value (CONF:1098-30334)
  9. SHOULD contain [0..*] author such that it (CONF:1098-31667)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  10. If value/@xsi:type="CD", the value SHOULD contain a code from SNOMED CT (CodeSystem: 2.16.840.1.113883.6.96) (CONF:1098-32433)

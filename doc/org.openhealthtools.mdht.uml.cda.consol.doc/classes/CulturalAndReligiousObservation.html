Cultural And Religious Observation

[Observation: templateId 2.16.840.1.113883.10.20.22.4.111]

This template represents a patient's spiritual, religious, and cultural belief practices, such as a kosher diet or fasting ritual. religiousAffiliationCode in the document header captures only the patient's religious affiliation.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-27926, CONF:1098-27927 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.111"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-27924)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-27925)
  4. SHALL contain at least one [1..*] id (CONF:1098-27928)
  5. SHALL contain exactly one [1..1] code (CONF:1098-27929)/@code="75281-6" Personal belief (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-27930, CONF:1098-27931)
  6. SHALL contain exactly one [1..1] statusCode (CONF:1098-27936)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-27937)
  7. SHALL contain exactly one [1..1] value (CONF:1098-28442)
  8. If value/@xsi:type="CD", the value SHALL contain a code from SNOMED CT (CodeSystem: 2.16.840.1.113883.6.96) (CONF:1098-32487)

Cultural And Religious Observation example 

<observation xmlns="urn:hl7-org:v3" classCode="OBS" moodCode="EVN">
  <templateId root="2.16.840.1.113883.10.20.22.4.111"/>
  <id root="MDHT" extension="848016089"/>
  <code code="75281-6" codeSystem="2.16.840.1.113883.6.1" codeSystemName="LOINC" displayName="Personal belief"/>
  <text>Text Value</text>
  <statusCode code="completed"/>
</observation>

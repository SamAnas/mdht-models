Sensory Status

[Observation: templateId 2.16.840.1.113883.10.20.22.4.127]

This template represents a patient's sensory or speech ability. It may contain an assessment scale observations related to the sensory or speech ability.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-27959, CONF:1098-27960 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.127"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-31017)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-31018)
  4. SHALL contain exactly one [1..1] code, which SHOULD be selected from ValueSet Sensory Status Problem Type 2.16.840.1.113883.11.20.9.50 DYNAMIC (CONF:1098-27962)
  5. SHALL contain exactly one [1..1] statusCode (CONF:1098-31437)/@code="completed" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-31438)
  6. SHALL contain exactly one [1..1] effectiveTime (CONF:1098-31441)
  7. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHOULD be selected from ValueSet Mental and Functional Status Response 2.16.840.1.113883.11.20.9.44 DYNAMIC (CONF:1098-27974)
  8. SHOULD contain [0..1] author such that it (CONF:1098-31439)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  9. MAY contain [0..*] entryRelationship such that it (CONF:1098-27984, CONF:1098-27985, CONF:1098-27986)
    1. Contains @typeCode="COMP" COMP
    2. Conforms to Assessment Scale Observation (templateId: 2.16.840.1.113883.10.20.22.4.69)

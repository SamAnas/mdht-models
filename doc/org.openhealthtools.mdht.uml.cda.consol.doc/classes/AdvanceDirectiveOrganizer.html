Advance Directive Organizer (V2)

[Organizer: templateId 2.16.840.1.113883.10.20.22.4.108]

This clinical statement groups a set of advance directive observations.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-28412, CONF:1198-28413, CONF:1198-32876 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.108"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL contain exactly one [1..1] @classCode="CLUSTER" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1198-28410)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1198-28411)
  4. SHALL contain at least one [1..*] id (CONF:1198-28414)
  5. SHALL contain exactly one [1..1] code (CONF:1198-28415)/@code="45473-6" Advance directive - living will (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-31230, CONF:1198-31231)
  6. SHALL contain exactly one [1..1] statusCode (CONF:1198-28418)/@code="completed" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1198-31346)
  7. SHOULD contain [0..*] author such that it (CONF:1198-32407)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  8. SHALL contain [1..*] component such that it (CONF:1198-28420, CONF:1198-28421)
    1. Conforms to Advance Directive Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.48:2015-08-01)

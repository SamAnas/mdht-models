Problem Section (V3) (entries required)

[Section: templateId 2.16.840.1.113883.10.20.22.2.5.1]

This section lists and describes all relevant clinical problems at the time the document is generated. At a minimum, all pertinent current and historical problems should be listed. Overall health status may be represented in this section.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-9179, CONF:1198-10441, CONF:1198-32510 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.5.1"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Problem Section Entries Optional2 template (templateId: 2.16.840.1.113883.10.20.22.2.5:2015-08-01)
  3. MAY contain zero or one [0..1] @nullFlavor="NI" (CONF:1198-32864)
  4. SHALL contain exactly one [1..1] code (CONF:1198-15409)/@code="11450-4" Problem List (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-15410, CONF:1198-31142)
  5. SHALL contain exactly one [1..1] title (CONF:1198-9181)
  6. SHALL contain exactly one [1..1] text (CONF:1198-9182)
  7. If section/@nullFlavor is not present, SHALL contain [1..*] entry such that it (CONF:1198-9183, CONF:1198-15506)

    If section/@nullFlavor is not present:

    1. Conforms to Problem Concern Act2 (templateId: 2.16.840.1.113883.10.20.22.4.3:2015-08-01)
  8. If section/@nullFlavor is not present, MAY contain [0..1] entry such that it (CONF:1198-30479, CONF:1198-30480)
    1. Conforms to Health Status Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.5:2014-06-09)

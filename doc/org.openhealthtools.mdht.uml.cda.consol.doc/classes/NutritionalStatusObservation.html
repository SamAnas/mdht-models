Nutritional Status Observation

[Observation: templateId 2.16.840.1.113883.10.20.22.4.124]

This template describes the overall nutritional status of the patient including findings related to nutritional status.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-29843, CONF:1098-29844 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.124"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-29841)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-29842)
  4. SHALL contain at least one [1..*] id (CONF:1098-29845)
  5. SHALL contain exactly one [1..1] code (CONF:1098-29846)/@code="75305-3" Nutrition status (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-29897, CONF:1098-29898)
  6. SHALL contain exactly one [1..1] statusCode (CONF:1098-29852)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-29853)
  7. SHALL contain exactly one [1..1] effectiveTime (CONF:1098-31867)
  8. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHOULD be selected from ValueSet Nutritional Status 2.16.840.1.113883.1.11.20.2.7 DYNAMIC (CONF:1098-29854)
  9. SHALL contain [1..*] entryRelationship such that it (CONF:1098-30323, CONF:1098-30335, CONF:1098-30336)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Nutrition Assessment (templateId: 2.16.840.1.113883.10.20.22.4.138)

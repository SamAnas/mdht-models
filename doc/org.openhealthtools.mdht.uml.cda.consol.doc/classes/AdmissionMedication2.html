Admission Medication2

[Act: templateId 2.16.840.1.113883.10.20.22.4.36]

This template represents the medications taken by the patient prior to and at the time of admission.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-16758, CONF:1098-16759, CONF:1098-32524 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.36"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Admission Medication template (templateId: 2.16.840.1.113883.10.20.22.4.36)
  3. SHALL contain exactly one [1..1] code (CONF:1098-15518)/@code="42346-7" Medications on Admission (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-15519, CONF:1098-32152)
  4. SHALL contain [1..*] entryRelationship such that it (CONF:1098-7701, CONF:1098-7702, CONF:1098-15520)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Medication Activity2 (templateId: 2.16.840.1.113883.10.20.22.4.16:2014-06-09)

Postprocedure Diagnosis Section (V3) - conformance rules

[Section: templateId 2.16.840.1.113883.10.20.22.2.36]

The Postprocedure Diagnosis Section records the diagnosis or diagnoses discovered or confirmed during the procedure. Often it is the same as the preprocedure diagnosis or indication.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-8167, CONF:1198-10438, CONF:1198-32550 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.36"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Postprocedure Diagnosis Section template (templateId: 2.16.840.1.113883.10.20.22.2.36)
  3. SHALL contain exactly one [1..1] code (CONF:1198-15403)/@code="59769-0" Postprocedure Diagnosis (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-15404, CONF:1198-30862)
  4. If section/@nullFlavor is not present, SHOULD contain [0..1] entry such that it (CONF:1198-8762, CONF:1198-15503)
    1. Conforms to Postprocedure Diagnosis2 (templateId: 2.16.840.1.113883.10.20.22.4.51:2015-08-01)
  1. SHALL conform to Postprocedure Diagnosis Section template (templateId: 2.16.840.1.113883.10.20.22.2.36)
  2. SHALL contain exactly one [1..1] code (CONF:1198-15403)/@code="59769-0" Postprocedure Diagnosis (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-15404, CONF:1198-30862)
  3. SHALL contain exactly one [1..1] title (CONF:8170, R2.1=CONF:1198-8170)
  4. SHALL contain exactly one [1..1] text (CONF:8171, R2.1=CONF:1198-8171)
  5. If section/@nullFlavor is not present, SHOULD contain [0..1] entry such that it (CONF:1198-8762, CONF:1198-15503)
    1. Conforms to Postprocedure Diagnosis2 (templateId: 2.16.840.1.113883.10.20.22.4.51:2015-08-01)

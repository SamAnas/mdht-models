Mental Status Observation - conformance rules

[Observation: templateId 2.16.840.1.113883.10.20.22.4.125]

This template represents observations relating intellectual and mental powers and state of mind. Mental status observations in a clinical note often have a psychological focus (e.g., level of consciousness, mood, anxiety level, reasoning ability).

  1. SHALL contain exactly one [1..1] templateId ( CONF:29186, CONF:29187 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.125"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:29182)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:29183)
  4. SHALL contain at least one [1..*] id (CONF:29188)
  5. SHALL contain exactly one [1..1] code, which SHOULD be selected from ValueSet Mental Status Observation Type 2.16.840.1.113883.11.20.9.43 DYNAMIC (CONF:29189)
  6. SHALL contain exactly one [1..1] statusCode (CONF:29194)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:29195)
  7. SHALL contain exactly one [1..1] effectiveTime (CONF:29196)
  8. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHOULD be selected from ValueSet Mental and Functional Status Response 2.16.840.1.113883.11.20.9.44 DYNAMIC (CONF:29202)
  9. SHOULD contain [0..*] author such that it (CONF:31435)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  10. MAY contain [0..*] entryRelationship such that it (CONF:29207, CONF:29208, CONF:29209)
    1. Contains @typeCode="COMP" COMP
    2. Conforms to Assessment Scale Observation (templateId: 2.16.840.1.113883.10.20.22.4.69)

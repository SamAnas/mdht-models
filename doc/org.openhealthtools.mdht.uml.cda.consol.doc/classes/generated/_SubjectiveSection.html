Subjective Section - conformance rules

[Section: templateId 2.16.840.1.113883.10.20.21.2.2]

The Subjective section describes in a narrative format the patient's current condition and/or interval changes as reported by the patient or by the patient's guardian or another informant.

  1. SHALL contain exactly one [1..1] templateId ( CONF:7873, CONF:10470 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.21.2.2"
  2. SHALL contain exactly one [1..1] code (CONF:15437)/@code="61150-9" Subjective (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:15438)
  3. SHALL contain exactly one [1..1] text (CONF:7876)
  4. SHALL contain exactly one [1..1] title (CONF:7875)

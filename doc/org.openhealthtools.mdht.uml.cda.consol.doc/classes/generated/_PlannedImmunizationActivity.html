Planned Immunization Activity - conformance rules

[SubstanceAdministration: templateId 2.16.840.1.113883.10.20.22.4.120]

This template represents planned immunizations. Planned Immunization Activity is very similar to Planned Medication Activity with some key differences, for example, the drug code system is constrained to CVX codes. The priority of the immunization activity to the patient and provider is communicated through Priority Preference. The effectiveTime indicates the time when the immunization activity is intended to take place and authorTime indicates when the documentation of the plan occurred.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-32098, CONF:1098-32099 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.120"
  2. SHALL contain exactly one [1..1] @classCode="SBADM" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-32091)
  3. SHALL contain exactly one [1..1] @moodCode, which SHALL be selected from ValueSet Planned moodCode (SubstanceAdministration/Supply) 2.16.840.1.113883.11.20.9.24 STATIC 2014-09-01 (CONF:1098-32097)
  4. SHALL contain at least one [1..*] id (CONF:1098-32100)
  5. SHALL contain exactly one [1..1] statusCode (CONF:1098-32101)/@code="active" Active (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-32102)
  6. SHALL contain exactly one [1..1] effectiveTime (CONF:1098-32103)

    The effectiveTime in a planned immunization activity represents the time that the immunization activity should occur.

  7. MAY contain zero or one [0..1] repeatNumber (CONF:1098-32126)

    In a Planned Immunization Activity, repeatNumber defines the number of allowed administrations. For example, a repeatNumber of "3" means that the substance can be administered up to 3 times.

  8. MAY contain zero or one [0..1] routeCode, which SHALL be selected from ValueSet Medication Route FDA Value Set 2.16.840.1.113883.3.88.12.3221.8.7 DYNAMIC (CONF:1098-32127)

  10. MAY contain zero or one [0..1] doseQuantity (CONF:1098-32129)
  11. SHALL contain exactly one [1..1] consumable such that it (CONF:1098-32131)
    1. SHALL contain exactly one [1..1] manufacturedProduct, where its type is Immunization Medication Information2 (CONF:1098-32132)
  12. MAY contain [0..*] performer such that it (CONF:1098-32104)

    2. Conforms to Performer2 from CDA
  13. MAY contain [0..*] author such that it (CONF:1098-32105)

    The author in a planned immunization activity represents the clinician who is requesting or planning the immunization activity.

    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  14. MAY contain [0..*] entryRelationship such that it (CONF:1098-32108, CONF:1098-32109, CONF:1098-32110)

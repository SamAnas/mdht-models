Medication Supply Order2 - conformance rules

[Supply: templateId 2.16.840.1.113883.10.20.22.4.17]

This template records the intent to supply a patient with medications.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-7429, CONF:1098-10507, CONF:1098-32578 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.17"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Medication Supply Order template (templateId: 2.16.840.1.113883.10.20.22.4.17)
  3. SHALL contain exactly one [1..1] statusCode (CONF:1098-7432), which SHALL be selected from ValueSet ActStatus 2.16.840.1.113883.1.11.159331 DYNAMIC (CONF:1098-32362)
  4. SHOULD contain zero or one [0..1] repeatNumber (CONF:1098-7434)

    In "INT" (intent) mood, the repeatNumber defines the number of allowed administrations. For example, a repeatNumber of "3" means that the substance can be administered up to 3 times. In "EVN" (event) mood, the repeatNumber is the number of occurrences. For example, a repeatNumber of "3" in a substance administration event means that the current administration is the 3rd in a series.

  5. MAY contain [0..1] entryRelationship such that it (CONF:1098-7442, CONF:1098-7444, CONF:1098-31391)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Instruction2 (templateId: 2.16.840.1.113883.10.20.22.4.20:2014-06-09)
  6. MAY contain zero or one [0..1] product such that it SHALL contain exactly one [1..1] Medication Information (V2) (identifier: urn:hl7ii:2.16.840.1.113883.10.20.22.4.23:2014-06-09) (CONF:1098-7439, CONF:1098-16093)
  7. MAY contain zero or one [0..1] product such that it SHALL contain exactly one [1..1] Immunization Medication Information (V2) (identifier: urn:hl7ii:2.16.840.1.113883.10.20.22.4.54:2014-06-09) (CONF:1098-9334, CONF:1098-31695)
  8. A supply act SHALL contain one product/Medication Information (V2) or one product/Immunization Medication Information (V2) template (CONF:1098-16870)
  9. entryRelationship with target entry Instruction2 if present, SHALL contain exactly one [1..1] @inversionInd="true" True (CONF:1098-7445)
  1. SHALL conform to Medication Supply Order template (templateId: 2.16.840.1.113883.10.20.22.4.17)
  2. SHALL contain exactly one [1..1] @classCode="SPLY" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:7427, R2.0=CONF:1098-7427)
  3. SHALL contain exactly one [1..1] @moodCode="INT" (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:7428, R2.0=CONF:1098-7428)
  4. SHALL contain at least one [1..*] id (Order Number) (CONF:7430, R2.0=CONF:1098-7430)
  5. SHALL contain exactly one [1..1] statusCode (CONF:1098-7432), which SHALL be selected from ValueSet ActStatus 2.16.840.1.113883.1.11.159331 DYNAMIC (CONF:1098-32362)
  6. SHOULD contain zero or one [0..1] effectiveTime (Order Expiration Date/Time) (CONF:15143, R2.0=CONF:1098-15143)
  7. SHOULD contain zero or one [0..1] repeatNumber (CONF:1098-7434)

    In "INT" (intent) mood, the repeatNumber defines the number of allowed administrations. For example, a repeatNumber of "3" means that the substance can be administered up to 3 times. In "EVN" (event) mood, the repeatNumber is the number of occurrences. For example, a repeatNumber of "3" in a substance administration event means that the current administration is the 3rd in a series.

  8. SHOULD contain zero or one [0..1] quantity (Quantity Ordered) (CONF:7436, R2.0=CONF:1098-7436)
  9. MAY contain [0..1] author (Ordering Provider) such that it (CONF:7438, R2.0=CONF:1098-7438)
    1. Conforms to Author from CDA
  10. MAY contain [0..1] entryRelationship such that it (CONF:1098-7442, CONF:1098-7444, CONF:1098-31391)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Instruction2 (templateId: 2.16.840.1.113883.10.20.22.4.20:2014-06-09)
  11. entryRelationship with target entry Instructions if present, SHALL contain exactly one [1..1] @inversionInd="true" True (CONF:7445)
  12. effectiveTime SHALL contain exactly one [1..1] high (CONF:15144, R2.0=CONF:1098-15144)
  13. supply act SHALL contain one product/Medication Information or one product/Immunization Medication Information template (CONF:16870)
  14. MAY contain zero or one [0..1] product such that it SHALL contain exactly one [1..1] Medication Information (V2) (identifier: urn:hl7ii:2.16.840.1.113883.10.20.22.4.23:2014-06-09) (CONF:1098-7439, CONF:1098-16093)
  15. MAY contain zero or one [0..1] product such that it SHALL contain exactly one [1..1] Immunization Medication Information (V2) (identifier: urn:hl7ii:2.16.840.1.113883.10.20.22.4.54:2014-06-09) (CONF:1098-9334, CONF:1098-31695)
  16. A supply act SHALL contain one product/Medication Information (V2) or one product/Immunization Medication Information (V2) template (CONF:1098-16870)
  17. entryRelationship with target entry Instruction2 if present, SHALL contain exactly one [1..1] @inversionInd="true" True (CONF:1098-7445)

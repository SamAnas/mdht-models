Health Status Evaluations And Outcomes Section - conformance rules

[Section: templateId 2.16.840.1.113883.10.20.22.2.61]

This template represents observations regarding the outcome of care from the interventions used to treat the patient. These observations represent status, at points in time, related to established care plan goals and/or interventions.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-29578, CONF:1098-29579 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.61"
  2. MAY contain zero or one [0..1] @nullFlavor="NI" (CONF:1098-32821)
  3. SHALL contain exactly one [1..1] code (CONF:1098-29580)/@code="11383-7" Patient Problem Outcome (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-29581, CONF:1098-29582)
  4. SHALL contain exactly one [1..1] title (CONF:1098-29589)
  5. SHALL contain exactly one [1..1] text (CONF:1098-29590)
  6. If section/@nullFlavor is not present, SHALL contain [1..*] entry such that it (CONF:1098-31227, CONF:1098-31228)
    1. Conforms to Outcome Observation (templateId: 2.16.840.1.113883.10.20.22.4.144)

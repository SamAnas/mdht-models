Medications Section2 - conformance rules

[Section: templateId 2.16.840.1.113883.10.20.22.2.1.1]

The Medications Section contains a patient's current medications and pertinent medication history. At a minimum, the currently active medications are listed. An entire medication history is an option. The section can describe a patient's prescription and dispense history and information about intended drug monitoring. This section requires either an entry indicating the subject is not known to be on any medications or entries summarizing the subject's medications.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-7568, CONF:1098-10433, CONF:1098-32499 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.1.1"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Medications Section Entries Optional2 template (templateId: 2.16.840.1.113883.10.20.22.2.1:2014-06-09)
  3. MAY contain zero or one [0..1] @nullFlavor="NI" (CONF:1098-32845)
  4. SHALL contain exactly one [1..1] code (CONF:1098-15387)/@code="10160-0" History of medication use (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-15388, CONF:1098-30825)
  5. SHALL contain exactly one [1..1] title (CONF:1098-7570)
  6. SHALL contain exactly one [1..1] text (CONF:1098-7571)
  7. If section/@nullFlavor is not present, SHALL contain [1..*] entry such that it (CONF:1098-7572, CONF:1098-10077)

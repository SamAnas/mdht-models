Wound Characteristic

[Observation: templateId 2.16.840.1.113883.10.20.22.4.134]

This template represents characteristics of a wound (e.g., integrity of suture line, odor, erythema).

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-29940, CONF:1098-29941 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.134"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-29938)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-29939)
  4. SHALL contain at least one [1..*] id (CONF:1098-29942)
  5. SHALL contain exactly one [1..1] code (CONF:1098-29943)/@code="ASSERTION" assertion (CodeSystem: 2.16.840.1.113883.5.4 HL7ActCode) (CONF:1098-31540, CONF:1098-31541)
  6. SHALL contain exactly one [1..1] statusCode/@code="completed" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-29944)
  7. SHALL contain exactly one [1..1] effectiveTime (CONF:1098-29946)
  8. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHALL be selected from ValueSet Wound Characteristic 2.16.840.1.113883.11.20.9.58 DYNAMIC (CONF:1098-29947)

<observation xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns="urn:hl7-org:v3" classCode="OBS" moodCode="EVN">
  <templateId root="2.16.840.1.113883.10.20.22.4.134"/>
  <id root="MDHT" extension="666728342"/>
  <code code="ASSERTION" codeSystem="2.16.840.1.113883.5.4" codeSystemName="HL7ActCode" displayName="assertion"/>
  <text>Text Value</text>
  <statusCode code="completed"/>
  <effectiveTime>
    <low value="2017"/>
    <high value="2017"/>
  </effectiveTime>
  <value xsi:type="CD" code="374199762"/>
</observation>

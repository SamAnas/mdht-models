Substance Or Device Allergy Observation

[Observation: templateId 2.16.840.1.113883.10.20.24.3.90]

This clinical statement represents that an allergy or adverse reaction to a substance or device exists or does not exist. The agent that is the cause of the allergy or adverse reaction is represented as a participant in the observation.

  1. SHALL contain exactly one [1..1] templateId ( CONF:16305, CONF:16306 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.24.3.90"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:16303, R2.0=CONF:1098-16303)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:16304, R2.0=CONF:1098-16304)
  4. SHALL contain at least one [1..*] id (CONF:16307, R2.0=CONF:1098-16307)
  5. SHALL contain exactly one [1..1] code (CONF:16345)/@code="ASSERTION" Assertion (CodeSystem: 2.16.840.1.113883.5.4 HL7ActCode) (CONF:16346)
  6. SHALL contain exactly one [1..1] statusCode (CONF:16308, R2.0=CONF:1098-16308)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:26354, R2.0=CONF:1098-26354)
  7. SHALL contain exactly one [1..1] effectiveTime (CONF:16309)
  8. SHALL contain exactly one [1..1] value with @xsi:type="CD" (CONF:16312, R2.0=CONF:1098-16312), which SHALL be selected from ValueSet Allergy/Adverse Event Type 2.16.840.1.113883.3.88.12.3221.6.2 DYNAMIC (CONF:16317, R2.0=CONF:1098-16317)
  9. MAY contain [0..1] entryRelationship such that it (CONF:16333, CONF:16335, CONF:16336, R2.0=CONF:1098-16333, R2.0=CONF:1098-16335, R2.0=CONF:1098-16336)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Allergy Status Observation (templateId: 2.16.840.1.113883.10.20.22.4.28)
  10. SHOULD contain [0..*] entryRelationship such that it (CONF:16337, CONF:16339, CONF:16340)
    1. Contains @typeCode="MFST" MFST
    2. Conforms to Reaction Observation (templateId: 2.16.840.1.113883.10.20.22.4.9)
  11. MAY contain [0..1] entryRelationship such that it (CONF:16341, CONF:16342, CONF:16344)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Severity Observation (templateId: 2.16.840.1.113883.10.20.22.4.8)
  12. SHOULD contain zero or more [0..*] participant such that it (CONF:16318)
    1. SHALL contain exactly one [1..1] @typeCode="CSM" (CONF:16319, R2.0=CONF:1098-16319)
    2. SHALL contain exactly one [1..1] participantRole such that it (CONF:16320)
      1. SHALL contain exactly one [1..1] @classCode="MANU" (CONF:16321, R2.0=CONF:1098-16321)
      2. SHALL contain exactly one [1..1] playingEntity such that it (CONF:16322)
        1. SHALL contain exactly one [1..1] @classCode="MMAT" (CONF:16323, R2.0=CONF:1098-16323)
        2. SHALL contain exactly one [1..1] code (CONF:16324)
        3. code/originalText/reference/@value SHALL begin with a '#' and SHALL point to its corresponding narrative (using the approach defined in CDA Release 2, section 4.3.5.1 (CONF:16329)
        4. In an allergy to a specific medication the code SHALL be selected from the ValueSet 2.16.840.1.113883.3.88.12.80.16 Medication Brand Name DYNAMIC or the ValueSet 2.16.840.1.113883.3.88.12.80.17 Medication Clinical Drug DYNAMIC (CONF:16325)
        5. In an allergy to a class of medications the code SHALL be selected from the ValueSet 2.16.840.1.113883.3.88.12.80.18 Medication Drug Class DYNAMIC (CONF:16331)
        6. In an allergy to a food or other substance the code SHALL be selected from the ValueSet 2.16.840.1.113883.3.88.12.80.20 Ingredient Name DYNAMIC (CONF:16332)
        7. reference, if present, SHOULD contain zero or one [0..1] @value (CONF:16328)
        8. originalText, if present, SHOULD contain zero or one [0..1] reference (CONF:16327)
        9. code MAY be selected from ValueSet Substance-Reactant for Intolerance 2.16.840.1.113762.1.4.1010.1 DYNAMIC. Note: Value set intentionally defined as a GROUPING made up of: Value Set: Medication Drug Class (2.16.840.1.113883.3.88.12.80.18) (NDFRT drug class codes); Value Set: Clinical Drug Ingredient (2.16.840.1.113762.1.4.1010.7) (RxNORM ingredient codes); Value Set: Unique Ingredient Identifier - Complete Set (2.16.840.1.113883.3.88.12.80.20) (UNII ingredient codes); Value Set: Substance Other Than Clinical Drug (2.16.840.1.113762.1.4.1010.9) (SNOMED CT substance codes). (CONF:16324, R2.0=CONF:1098-16324)
  13. entryRelationship with target entry Allergy Status Observation SHALL contain exactly one [1..1] @inversionInd="true" True (CONF:16334, R2.0=CONF:1098-16334)
  14. entryRelationship with target entry Reaction Observation SHALL contain exactly one [1..1] @inversionInd="true" True (CONF:16338)
  15. entryRelationship with target entry Severity Observation SHALL contain exactly one [1..1] @inversionInd="true" True (CONF:16343)

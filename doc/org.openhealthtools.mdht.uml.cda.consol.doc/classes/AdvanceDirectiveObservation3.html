Advance Directive Observation (V4)

[Observation: templateId 2.16.840.1.113883.10.20.22.4.48]

  1. SHALL contain exactly one [1..1] templateId such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.48"
    2. SHALL contain exactly one [1..1] @extension="2017-05-01"
  2. SHALL conform to Advance Directive Observation2 template (templateId: 2.16.840.1.113883.10.20.22.4.48:2015-08-01)
  3. SHALL contain exactly one [1..1] value with @xsi:type="CD" (CONF:1198-30804), which SHALL be selected from ValueSet Advance Directives - Types of Content 2.16.840.1.113883.11.20.9.69.5 STATIC
  4. SHALL contain exactly one [1..1] code, which SHOULD be selected from ValueSet Advance Directives - Kinds of 2.16.840.1.113883.11.20.9.69.4 STATIC (CONF:1198-8651)

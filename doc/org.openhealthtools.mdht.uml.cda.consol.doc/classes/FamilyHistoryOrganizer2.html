Family History Organizer (V3)

[Organizer: templateId 2.16.840.1.113883.10.20.22.4.45]

The Family History Organizer associates a set of observations with a family member. For example, the Family History Organizer can group a set of observations about the patient's father.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-8604, CONF:1198-10497, CONF:1198-32606 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.45"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Family History Organizer template (templateId: 2.16.840.1.113883.10.20.22.4.45)
  3. SHALL contain at least one [1..*] id (CONF:1198-32485)
  4. SHALL contain [1..*] component such that it (CONF:1198-32428, CONF:1198-32429)
    1. Conforms to Family History Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.46:2015-08-01)

Advance Directives Section (entries required) (V3)

[Section: templateId 2.16.840.1.113883.10.20.22.2.21.1]

This section contains data defining the patient's advance directives and any reference to supporting documentation, including living wills, healthcare proxies, and CPR and resuscitation status. If the referenced documents are available, they can be included in the exchange package. The most recent directives are required, if known, and should be listed in as much detail as possible. This section differentiates between "advance directives" and "advance directive documents". The former is the directions to be followed whereas the latter refers to a legal document containing those directions.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-30227, CONF:1198-30228, CONF:1198-32512 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.21.1"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Advance Directives Section Entries Optional2 template (templateId: 2.16.840.1.113883.10.20.22.2.21:2015-08-01)
  3. MAY contain zero or one [0..1] @nullFlavor="NI" (CONF:1198-32800)
  4. SHALL contain exactly one [1..1] code (CONF:1198-32929)/@code="42348-3" Advance Directives (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-32930, CONF:1198-32931)
  5. SHALL contain exactly one [1..1] title (CONF:1198-32932)
  6. SHALL contain exactly one [1..1] text (CONF:1198-32933)
  7. If section/@nullFlavor is not present, contains [0..*] entry such that it
    1. Conforms to Advance Directive Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.48:2015-08-01)
  8. If section/@nullFlavor is not present, contains [0..*] entry such that it
    1. Conforms to Advance Directive Organizer (templateId: 2.16.840.1.113883.10.20.22.4.108:2015-08-01)
  9. If section/@nullFlavor is not present, SHALL contain at least one [1..*] entry such that it (CONF:1198-30235)
    1. MAY contain zero or one [0..1] observation, where its type is Advance Directive Observation (V3) (CONF:1198-30236)
    2. MAY contain zero or one [0..1] organizer, where its type is Advance Directive Organizer (V2) (CONF:1198-32420)
  10. If section/@nullFlavor is not present, this entry SHALL contain EITHER an Advance Directive Observation (V3) OR an Advance Directive Organizer (V2) (CONF:1198-32881)

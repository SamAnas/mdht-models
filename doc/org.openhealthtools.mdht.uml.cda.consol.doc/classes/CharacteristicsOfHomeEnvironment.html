Characteristics Of Home Environment

[Observation: templateId 2.16.840.1.113883.10.20.22.4.109]

This template represents the patient's home environment including, but not limited to, type of residence (trailer, single family home, assisted living), living arrangement (e.g., alone, with parents), and housing status (e.g., evicted, homeless, home owner).

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-27892, CONF:1098-27893 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.109"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-27890)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-27891)
  4. SHALL contain at least one [1..*] id (CONF:1098-27894)
  5. SHALL contain exactly one [1..1] code (CONF:1098-31352)/@code="75274-1" Characteristics of residence (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-31353, CONF:1098-31354)
  6. SHALL contain exactly one [1..1] statusCode (CONF:1098-27901)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-27902)
  7. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHOULD be selected from ValueSet Residence and Accommodation Type 2.16.840.1.113883.11.20.9.49 DYNAMIC (CONF:1098-28823)

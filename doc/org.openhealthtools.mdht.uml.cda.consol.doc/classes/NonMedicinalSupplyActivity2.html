Non Medicinal Supply Activity2

[Supply: templateId 2.16.840.1.113883.10.20.22.4.50]

This template represents equipment supplied to the patient (e.g., pumps, inhalers, wheelchairs). Devices applied to, or placed in, the patient are represented with the Product Instance entry contained within a Procedure Activity Procedure (V2) (identifier: urn:hl7ii: 2.16.840.1.113883.10.20.22.4.14:2014-06-09)

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-8747, CONF:1098-10509, CONF:1098-32514 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.50"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Non Medicinal Supply Activity template (templateId: 2.16.840.1.113883.10.20.22.4.50)
  3. SHALL contain exactly one [1..1] statusCode (CONF:1098-8749), which SHALL be selected from ValueSet ActStatus 2.16.840.1.113883.1.11.159331 DYNAMIC (CONF:1098-32363)
  4. MAY contain [0..1] entryRelationship such that it (CONF:1098-30277, CONF:1098-30278, CONF:1098-31393)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Instruction2 (templateId: 2.16.840.1.113883.10.20.22.4.20:2014-06-09)
  5. MAY contain [0..1] participant such that it (CONF:1098-8752, CONF:1098-15900)
    1. Conforms to Product Instance (templateId: 2.16.840.1.113883.10.20.22.4.37)
  6. entryRelationship with target entry Instruction2 if present, SHALL contain exactly one [1..1] @inversionInd="true" (CONF:1098-30279)

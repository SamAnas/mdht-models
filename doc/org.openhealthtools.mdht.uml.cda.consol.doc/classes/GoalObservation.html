Goal Observation

[Observation: templateId 2.16.840.1.113883.10.20.22.4.121]

This template represents a patient health goal. A Goal Observation template may have related components that are acts, encounters, observations, procedures, substance administrations, or supplies. A goal may be a patient or provider goal. If the author is the recordTarget (patient), this is a patient goal. If the author is a provider, this is a provider goal. If both patient and provider are authors, this is a negotiated goal. If no author is present, it is assumed the document or section author(s) is the author of this goal. A goal usually has a related health concern and/or risk. A goal may have components consisting of other goals (milestones). These milestones are related to the overall goal through entryRelationships.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-8583, CONF:1098-10512 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.121"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-30418)
  3. SHALL contain exactly one [1..1] @moodCode="GOL" (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-30419)
  4. SHALL contain at least one [1..*] id (CONF:1098-32332)
  5. SHALL contain exactly one [1..1] code, which SHOULD be selected from (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-30784)
  6. SHALL contain exactly one [1..1] statusCode (CONF:1098-32333)/@code="active" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-32334)
  7. SHOULD contain zero or one [0..1] effectiveTime (CONF:1098-32335)
  8. MAY contain zero or one [0..1] value (CONF:1098-32743)
  9. SHOULD contain [0..*] author such that it (CONF:1098-30995, DSTU:867)

    1. If the author is the recordTarget (patient), this is a patient goal. If the author is a provider, this is a provider goal. If both patient and provider are authors, this is a negotiated goal. If no author is present, it is assumed the document or section author(s) is the author of this goal.

    If the author is set to the recordTarget (patient), this is a patient goal. If the author is set to a provider, this is a provider goal. If both patient and provider are set as authors, this is a negotiated goal.

    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  10. MAY contain [0..*] entryRelationship such that it (CONF:1098-30701, CONF:1098-30702, CONF:1098-30703)

    The following entryRelationship represents the relationship between a Goal Observation and a Health Concern Act (Goal Observation REFERS TO Health Concern Act). As Health Concern Act is already defined in Health Concerns Section, rather than clone the whole Health Concern Act template, an Entry Reference may be used in entryRelationship to refer the template.

    1. Contains @typeCode="REFR" REFR
    2. Conforms to Entry Reference (templateId: 2.16.840.1.113883.10.20.22.4.122)
  11. MAY contain [0..*] entryRelationship such that it (CONF:1098-30704, CONF:1098-30705, CONF:1098-32879)

    The following entryRelationship represents a planned component of the goal such as Planned Encounter (V2), Planned Observation (V2), Planned Procedure (V2), Planned Medication Activity (V2), Planned Supply (V2), Planned Act (V2) or Planned Immunization Activity. Because these entries are already described in the Interventions Section of the CDA document instance, rather than repeating the full content of the entries, the Entry Reference template may be used to reference the entries.

    1. Contains @typeCode="COMP" COMP
    2. Conforms to Entry Reference (templateId: 2.16.840.1.113883.10.20.22.4.122)
  12. SHOULD contain [0..1] entryRelationship such that it (CONF:1098-30785, CONF:1098-30786, CONF:1098-30787)

    The following entryRelationship represents the priority that the patient or a provider puts on the goal.

    1. Contains @typeCode="REFR" REFR
    2. Conforms to Priority Preference (templateId: 2.16.840.1.113883.10.20.22.4.143)
  13. MAY contain [0..*] entryRelationship such that it (CONF:1098-31448, CONF:1098-31449, CONF:1098-32880)

    The following entryRelationship represents the relationship between two Goal Observations where the target is a component of the source (Goal Observation HAS COMPONENT Goal Observation). The component goal (target) is a Milestone.

    1. Contains @typeCode="COMP" COMP
    2. Conforms to Goal Observation (templateId: 2.16.840.1.113883.10.20.22.4.121)
  14. MAY contain [0..*] entryRelationship such that it (CONF:1098-31559, CONF:1098-31560, CONF:1098-31588)

    Where a Goal Observation needs to reference another entry already described in the CDA document instance, rather than repeating the full content of the entry, the Entry Reference template may be used to reference this entry.

    1. Contains @typeCode="REFR" REFR
    2. Conforms to Entry Reference (templateId: 2.16.840.1.113883.10.20.22.4.122)
  15. MAY contain zero or more [0..*] reference such that it (CONF:1098-32754)
    1. SHALL contain exactly one [1..1] @typeCode="REFR" (CONF:1098-32755)
    2. SHALL contain exactly one [1..1] externalDocument, where its type is External Document Reference (CONF:1098-32756)

Discharge Medications Section (entries required) (V3)

[Section: templateId 2.16.840.1.113883.10.20.22.2.11.1]

This section contains the medications the patient is intended to take or stop after discharge. Current, active medications must be listed. The section may also include a patient's prescription history and indicate the source of the medication list.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-7822, CONF:1198-10397, CONF:1198-32562 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.11.1"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Discharge Medications Section Entries Optional2 template (templateId: 2.16.840.1.113883.10.20.22.2.11:2015-08-01)
  3. MAY contain zero or one [0..1] @nullFlavor="NI" (CONF:1198-32812)
  4. SHALL contain exactly one [1..1] code (CONF:1198-15361)/@code="10183-2" Hospital Discharge Medications (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-15362, CONF:1198-32145)
  5. SHALL contain exactly one [1..1] title (CONF:1198-7824)
  6. SHALL contain exactly one [1..1] text (CONF:1198-7825)
  7. If section/@nullFlavor is not present, SHALL contain [1..*] entry such that it (CONF:1198-7826, CONF:1198-15491, DSTU:820)

    If section/@nullFlavor is not present:

    1. Conforms to Discharge Medication2 (templateId: 2.16.840.1.113883.10.20.22.4.35:2016-03-01)

Self Care Activities

[Observation: templateId 2.16.840.1.113883.10.20.22.4.128]

This template represents a patient's daily self-care ability. These activities are called Activities of Daily Living (ADL) and Instrumental Activities of Daily Living (IADL). ADLs involve caring for and moving of the body (e.g., dressing, bathing, eating). IADLs support an independent life style (e.g., cooking, managing medications, driving, shopping).

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-28190, CONF:1098-28457 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.128"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-31389)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-31390)
  4. SHALL contain exactly one [1..1] code, which SHOULD be selected from ValueSet ADL Result Type 2.16.840.1.113883.11.20.9.47 DYNAMIC (CONF:1098-28153)

    If more detailed ADL and IADL activities need to be recorded select the appropriate code from LOINC.

  5. SHALL contain exactly one [1..1] statusCode (CONF:1098-32490)/@code="completed" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-32491)
  6. SHALL contain exactly one [1..1] effectiveTime (CONF:1098-32492)
  7. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHOULD be selected from ValueSet Ability 2.16.840.1.113883.11.20.9.46 STATIC (CONF:1098-28042)
  8. SHOULD contain [0..*] author such that it (CONF:1098-32469)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)

Functional Status Observation2

[Observation: templateId 2.16.840.1.113883.10.20.22.4.67]

This template represents the patient's physical function (e.g., mobility status, instrumental activities of daily living, self-care status) and problems that limit function (dyspnea, dysphagia). The template may include assessment scale observations, identify supporting caregivers, and provide information about non-medicinal supplies. This template is used to represent physical or developmental function of all patient populations.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-13889, CONF:1098-13890, CONF:1098-32568 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.67"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Functional Status Result Observation template (templateId: 2.16.840.1.113883.10.20.22.4.67)
  3. SHALL contain exactly one [1..1] code (CONF:1098-13908)/@code="54522-8" Functional status (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-31522, CONF:1098-31523)
  4. contain zero or one [0..1] text
    1. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
  5. contains zero or more [0..*] interpretationCode
  6. contain zero or one [0..1] methodCode
  7. contain zero or one [0..1] targetSiteCode with @xsi:type="CE"
  8. MAY contain zero or more [0..*] referenceRange such that it (CONF:1098-13937)

    referenceRange could be used to represent normal or expected capability for the function being evaluated.

    1. contain exactly one [1..1] observationRange such that it
      1. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
  9. contain [0..1] author such that it
    1. Conforms to Author from CDA
  10. SHOULD contain [0..*] author such that it (CONF:1098-13936)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  11. MAY contain [0..1] entryRelationship such that it (CONF:1098-13892, CONF:1098-14596, CONF:1098-14218)
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Non Medicinal Supply Activity2 (templateId: 2.16.840.1.113883.10.20.22.4.50:2014-06-09)
  12. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire

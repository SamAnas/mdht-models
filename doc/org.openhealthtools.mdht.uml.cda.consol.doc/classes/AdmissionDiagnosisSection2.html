Admission Diagnosis Section (V3)

[Section: templateId 2.16.840.1.113883.10.20.22.2.43]

This section contains a narrative description of the problems or diagnoses identified by the clinician at the time of the patient's admission. This section may contain coded entries representing the admitting diagnoses.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-9930, CONF:1198-10391, CONF:1198-32563 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.43"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Hospital Admission Diagnosis Section template (templateId: 2.16.840.1.113883.10.20.22.2.43)
  3. SHALL contain exactly one [1..1] code (CONF:1198-15479)/@code="46241-6" Hospital Admission diagnosis (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-15480, CONF:1198-30865)
  4. If section/@nullFlavor is not present, SHOULD contain [0..1] entry such that it (CONF:1198-9934, CONF:1198-15481)
    1. Conforms to Hospital Admission Diagnosis2 (templateId: 2.16.840.1.113883.10.20.22.4.34:2015-08-01)

Tobacco Use

[Observation: templateId 2.16.840.1.113883.10.20.22.4.85]

This clinical statement represents a patient's tobacco use. All types of tobacco use are represented using the codes from the tobacco use and exposure - finding hierarchy in SNOMED CT.

  1. SHALL contain exactly one [1..1] templateId ( CONF:16566, CONF:16567 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.85"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:16558, R2.0=CONF:1098-16558)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:16559, R2.0=CONF:1098-16559)
  4. SHALL contain exactly one [1..1] code (CONF:16560)
  5. SHALL contain exactly one [1..1] statusCode (R2.0=CONF:1098-16561)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:16561, R2.0=CONF:1098-19118)
  6. SHALL contain exactly one [1..1] effectiveTime (CONF:16564)

    Note: The effectiveTime represents the biologically relevant time of the observation. Thus, an observation of "former smoker" will have an effectiveTime defining the time during which the patient has been a former smoker; an observation of "current smoker" will have an effectiveTime defining the time during which the patient has been a current smoker.

  7. SHALL contain exactly one [1..1] value with @xsi:type="CD" (CONF:16562, R2.0=CONF:1098-16562), which SHALL be selected from ValueSet Tobacco Use 2.16.840.1.113883.11.20.9.41 DYNAMIC (CONF:16563, R2.0=CONF:1098-16563)
  8. effectiveTime SHALL contain exactly one [1..1] low (CONF:16565, R2.0=CONF:1098-16565)

Tobacco Use example 

<observation xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns="urn:hl7-org:v3" classCode="OBS" moodCode="EVN">

  <text>Text Value</text>
  <statusCode code="completed"/>
  <effectiveTime>
    <low value="2017"/>
    <high value="2017"/>
  </effectiveTime>
  <value xsi:type="CD" code="1446097735"/>
</observation>

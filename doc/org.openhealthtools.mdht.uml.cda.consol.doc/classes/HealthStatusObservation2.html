Health Status Observation2

[Observation: templateId 2.16.840.1.113883.10.20.22.4.5]

This template represents information about the overall health status of the patient. To represent the impact of a specific problem or concern related to the patient's expected health outcome use the Prognosis Observation template 2.16.840.1.113883.10.20.22.4.113.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-16756, CONF:1098-16757, CONF:1098-32558 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.5"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Health Status Observation template (templateId: 2.16.840.1.113883.10.20.22.4.5)
  3. SHALL contain at least one [1..*] id (CONF:1098-32486)
  4. SHALL contain exactly one [1..1] code (CONF:1098-19143)/@code="11323-3" Health status (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-19144, CONF:1098-32161)
  5. contain zero or one [0..1] text
  6. SHALL contain exactly one [1..1] statusCode (CONF:1098-9074)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-19103)
  7. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
  8. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
  9. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire

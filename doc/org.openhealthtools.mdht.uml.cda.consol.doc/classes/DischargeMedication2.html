Discharge Medication (V3)

[Act: templateId 2.16.840.1.113883.10.20.22.4.35]

This template represents medications that the patient is intended to take (or stop) after discharge.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-16760, CONF:1098-16761, CONF:1098-32513 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.35"
    2. SHALL contain exactly one [1..1] @extension="2016-03-01"
  2. SHALL conform to Discharge Medication template (templateId: 2.16.840.1.113883.10.20.22.4.35)
  3. SHALL contain exactly one [1..1] code (CONF:1098-7691)/@code="10183-2" Hospital discharge medications (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1098-19161, DSTU:820, CONF:1098-32159)
  4. SHALL contain exactly one [1..1] statusCode (CONF:1098-32779)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-32780)
  5. SHALL contain [1..*] entryRelationship such that it (CONF:1098-7692, CONF:1098-7693, CONF:1098-15525)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Medication Activity2 (templateId: 2.16.840.1.113883.10.20.22.4.16:2014-06-09)

Medication Free Text Sig

[SubstanceAdministration: templateId 2.16.840.1.113883.10.20.22.4.147]

The template is available to explicitly identify the free text Sig within each medication. An example free text sig: Thyroxin 150 ug, take one tab by mouth every morning

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-32753, CONF:1198-32772 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.147"
  2. SHALL contain exactly one [1..1] @classCode="SBADM" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1198-32770)
  3. SHALL contain exactly one [1..1] @moodCode, which SHALL be selected from ValueSet MoodCodeEvnInt 2.16.840.1.113883.11.20.9.18 STATIC 2011-04-03 (CONF:1198-32771)

    Note: moodCode must match the parent substanceAdministration EVN or INT

  4. SHALL contain exactly one [1..1] code (CONF:1198-32775)/@code="76662-6" Instructions Medication (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-32780, CONF:1198-32781)
  5. SHALL contain exactly one [1..1] text (CONF:1198-32754)
  6. SHALL contain exactly one [1..1] consumable such that it (CONF:1198-32776)
    1. SHALL contain exactly one [1..1] manufacturedProduct such that it (CONF:1198-32777)
      1. SHALL contain exactly one [1..1] manufacturedLabeledDrug such that it (CONF:1198-32778)
        1. SHALL contain exactly one [1..1] @nullFlavor="NA" (CONF:1198-32779)
  7. reference/@value SHALL begin with a '#' and SHALL point to its corresponding narrative (using the approach defined in CDA Release 2, section 4.3.5.1) (CONF:1198-32774)

Medication Free Text Sig example 

<substanceadministration xmlns="urn:hl7-org:v3" classCode="SBADM">
  <templateId root="2.16.840.1.113883.10.20.22.4.147"/>
  <code code="76662-6" codeSystem="2.16.840.1.113883.6.1" codeSystemName="LOINC" displayName="Instructions Medication"/>
  <consumable>
    <manufacturedProduct>
      <manufacturedLabeledDrug nullFlavor="NA"/>
    </manufacturedProduct>
  </consumable>
</substanceadministration>

Substance Administered Act

[Act: templateId 2.16.840.1.113883.10.20.22.4.118]

This template represents the administration course in a series. The entryRelationship/sequenceNumber in the containing template shows the order of this particular administration in that medication series.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-31502, CONF:1098-31503 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.118"
  2. SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-31500)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-31501)
  4. SHALL contain at least one [1..*] id (CONF:1098-31504)
  5. SHALL contain exactly one [1..1] code (CONF:1098-31506)/@code="416118004" Administration (CodeSystem: 2.16.840.1.113883.6.96 SNOMEDCT) (CONF:1098-31507, CONF:1098-31508)
  6. SHALL contain exactly one [1..1] statusCode/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-31505)
  7. MAY contain zero or one [0..1] effectiveTime (CONF:1098-31509)

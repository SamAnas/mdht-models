Nutrition Recommendation

[Act: templateId 2.16.840.1.113883.10.20.22.4.130]

This template represents nutrition regimens (e.g., fluid restrictions, calorie minimum), interventions (e.g., NPO, nutritional supplements), and procedures (e.g., G-Tube by bolus, TPN by central line). It may also depict the need for nutrition education.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-30340, CONF:1098-30341 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.130"
  2. SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-30385)
  3. SHALL contain exactly one [1..1] @moodCode, which SHALL be selected from ValueSet Planned moodCode (Act/Encounter/Procedure) 2.16.840.1.113883.11.20.9.23 STATIC 2014-09-01 (CONF:1098-30386)
  4. SHALL contain exactly one [1..1] code, which SHOULD be selected from ValueSet Nutrition Recommendations 2.16.840.1.113883.1.11.20.2.9 DYNAMIC (CONF:1098-30342)
  5. SHALL contain exactly one [1..1] statusCode (CONF:1098-31697)/@code="active" Active (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-31698)
  6. SHOULD contain zero or one [0..1] effectiveTime (CONF:1098-31699)

    The effectiveTime indicates the time when the activity is intended to take place.

  7. MAY contain [0..*] entryRelationship such that it
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Planned Encounter2 (templateId: 2.16.840.1.113883.10.20.22.4.40:2014-06-09)
  8. MAY contain [0..*] entryRelationship such that it
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Planned Medication Activity2 (templateId: 2.16.840.1.113883.10.20.22.4.42:2014-06-09)
  9. MAY contain [0..*] entryRelationship such that it
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Planned Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.44:2014-06-09)
  10. MAY contain [0..*] entryRelationship such that it
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Planned Procedure2 (templateId: 2.16.840.1.113883.10.20.22.4.41:2014-06-09)
  11. MAY contain [0..*] entryRelationship such that it
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Planned Supply2 (templateId: 2.16.840.1.113883.10.20.22.4.43:2014-06-09)
  12. MAY contain [0..*] entryRelationship such that it
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Planned Act2 (templateId: 2.16.840.1.113883.10.20.22.4.39:2014-06-09)

Deceased Observation

[Observation: templateId 2.16.840.1.113883.10.20.22.4.79]

This clinical statement represents the observation that a patient has expired. It also represents the cause of death, indicated by an entryRelationship type of "CAUS".

  1. SHALL contain exactly one [1..1] templateId ( CONF:14871, CONF:14872 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.79"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:14851, R2.1=CONF:1198-14851)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:14852, R2.1=CONF:1198-14852)
  4. SHALL contain at least one [1..*] id (CONF:14873, R2.1=CONF:1198-14873)
  5. SHALL contain exactly one [1..1] code (CONF:14853)/@code="ASSERTION" Assertion (CodeSystem: 2.16.840.1.113883.5.4 HL7ActCode)
  6. SHALL contain exactly one [1..1] statusCode (CONF:14854, R2.1=CONF:1198-14854)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (R2.1=CONF:1198-19095)
  7. SHALL contain exactly one [1..1] effectiveTime (CONF:14855, R2.1=CONF:1198-14855)
  8. SHALL contain exactly one [1..1] value with @xsi:type="CD" (CONF:14857, R2.1=CONF:1198-14857)/@code="419099009" Dead (CodeSystem: 2.16.840.1.113883.6.96 SNOMEDCT) (CONF:15142, R2.1=CONF:1198-15142)
  9. SHOULD contain [0..1] entryRelationship such that it (CONF:14868, CONF:14875, CONF:14870)
    1. Contains @typeCode="CAUS" CAUS
    2. Conforms to Problem Observation (templateId: 2.16.840.1.113883.10.20.22.4.4)
  10. effectiveTime SHALL contain exactly one [1..1] low (CONF:14874, R2.1=CONF:1198-14874)

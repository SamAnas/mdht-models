Functional Status Organizer2

[Organizer: templateId 2.16.840.1.113883.10.20.22.4.66]

This template groups related functional status observations into categories (e.g., mobility, self-care).

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-14361, CONF:1098-14362, CONF:1098-32569 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.66"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Functional Status Result Organizer template (templateId: 2.16.840.1.113883.10.20.22.4.66)
  3. SHALL contain exactly one [1..1] code (CONF:1098-14364)

    The code selected should indicate the category that groups the contained functional status evaluation observations (e.g., mobility, self-care, communication).

    1. /code SHOULD be selected from ICF (codeSystem 2.16.840.1.113883.6.254) OR LOINC (2.16.840.1.113883.6.1) (CONF:1098-31417)
  4. SHALL contain exactly one [1..1] statusCode (CONF:1098-14358)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-31434)
  5. SHOULD contain [0..*] author such that it (CONF:1098-31585)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  6. SHALL contain [1..*] component such that it (CONF:1098-14359, CONF:1098-14368)
    1. Conforms to Functional Status Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.67:2014-06-09)
  7. SHALL contain [1..*] component such that it (CONF:1098-31432, CONF:1098-31433)
    1. Conforms to Self Care Activities (templateId: 2.16.840.1.113883.10.20.22.4.128)

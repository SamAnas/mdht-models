Advance Care Planning Intervention (V1)

[Procedure: templateId 2.16.840.1.113883.10.20.22.4.204]

  1. SHALL contain exactly one [1..1] templateId such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.204"
    2. SHALL contain exactly one [1..1] @extension="2017-05-01"
  2. SHALL contain exactly one [1..1] @classCode="PROC" (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (3332-32991)
  3. SHALL contain exactly one [1..1] @moodCode (3332), which SHALL be selected from ValueSet PlannedorCompletedMoodCode 2.16.840.1.113883.11.20.9.69.6 DYNAMIC (32995)
  4. SHALL contain at least one [1..*] id (3332-32993)
  5. SHALL contain exactly one [1..1] code (3332-32947), which SHALL be selected from ValueSet Advance Care Planning Services Grouping Value Set 2.16.840.1.113883.11.20.9.69.1 STATIC
  6. SHALL contain exactly one [1..1] statusCode (3332), which SHALL be selected from ValueSet ProcedureActStatusCode 2.16.840.1.113883.11.20.9.22 DYNAMIC (32969)
  7. SHALL contain exactly one [1..1] effectiveTime
  8. SHOULD contain zero or more [0..*] performer such that it (3332-33014)

    The performer records the person who is intended to complete the planned action, or the person who completed the action when moodCode=EVN. The performer/time element records when the performer is expected to complete the action, or when the performer completed the action if moodCode=EVN.

    1. SHOULD contain zero or one [0..1] time (3332-33015)
  9. SHALL contain [1..*] author such that it
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  10. MAY contain zero or more [0..*] entryRelationship such that it (3332-33016)
    1. SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1098-31485)
    2. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-31486)
    3. SHALL contain at least one [1..*] id (CONF:1098-31489) such that it

      The ID must equal another entry/id in the same document instance. Application Software must be responsible for resolving the identifier back to its original object and then rendering the information in the correct place in the containing section's narrative text.

      The ID must equal another entry/id in the same document instance. Application Software must be responsible for resolving the identifier back to its original object and then rendering the information in the correct place in the containing section's narrative text.

    4. SHALL contain exactly one [1..1] code (CONF:1098-31490)
    5. SHALL contain exactly one [1..1] statusCode (CONF:1098-31498)/@code="completed" (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1098-31499)
    1. Contains @typeCode="REFR" REFR

US Realm Header - Patient Generated Document (V2)

[ClinicalDocument: templateId 2.16.840.1.113883.10.20.29.1]

This template is designed to be used in conjunction with the US Realm Header (V3). It includes additional conformances which further constrain the US Realm Header (V3). The Patient Generated Document Header template is not a separate document type. The document body may contain any structured or unstructured content from C-CDA.

US Realm Header - Patient Generated Document (V2) Header Constraints

Mental Status Observation (V3)

[Observation: templateId 2.16.840.1.113883.10.20.22.4.74]

The Mental Status Observation template represents an observation about mental status that can come from a broad range of subjective and objective information (including measured data) to address those categories described in the Mental Status Section. See also Assessment Scale Observation for specific collections of observations that together yield a summary evaluation of a particular condition.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-14255, CONF:1198-14256, CONF:1198-32565 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.74"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Cognitive Status Result Observation template (templateId: 2.16.840.1.113883.10.20.22.4.74)
  3. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1198-14249)
  4. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1198-14250)
  5. SHALL contain at least one [1..*] id (CONF:1198-14257)
  6. SHALL contain exactly one [1..1] code (CONF:1198-14591)/@code="373930000" Cognitive function (CodeSystem: 2.16.840.1.113883.6.96 SNOMEDCT) (CONF:1198-32788, CONF:1198-32789)
  7. SHALL contain exactly one [1..1] statusCode (CONF:1198-14254)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1198-19092)
  8. SHALL contain exactly one [1..1] effectiveTime (CONF:1198-14261)
  9. SHALL contain exactly one [1..1] value (CONF:1198-14263)
  10. contain zero or one [0..1] text
  11. contains zero or more [0..*] interpretationCode
  12. contain zero or one [0..1] methodCode
  13. contain zero or one [0..1] targetSiteCode with @xsi:type="CE"
  14. SHOULD contain [0..*] author such that it (CONF:1198-14266)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  15. MAY contain [0..*] entryRelationship such that it (CONF:1198-14469, CONF:1198-14595, CONF:1198-14470)
    1. Contains @typeCode="COMP" COMP
    2. Conforms to Assessment Scale Observation (templateId: 2.16.840.1.113883.10.20.22.4.69)
  16. MAY contain zero or more [0..*] referenceRange such that it (CONF:1198-14267)

    The referenceRange could be used to represent normal or expected capability for the mental function being evaluated.

    1. contain exactly one [1..1] observationRange such that it
      1. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire

    1. Conforms to Author from CDA
  18. contains [0..*] entryRelationship such that it

  19. contains [0..*] entryRelationship such that it
    1. Conforms to Caregiver Characteristics (templateId: 2.16.840.1.113883.10.20.22.4.72)
  20. If value/@xsi:type="CD", SHOULD contain a code from SNOMED CT (CodeSystem: 2.16.840.1.113883.6.96) (CONF:1198-14271)

Problem Section (V3) (entries optional)

[Section: templateId 2.16.840.1.113883.10.20.22.2.5]

This section lists and describes all relevant clinical problems at the time the document is generated. At a minimum, all pertinent current and historical problems should be listed. Overall health status may be represented in this section.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-7877, CONF:1198-10440, CONF:1198-32511 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.5"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Problem Section Entries Optional template (templateId: 2.16.840.1.113883.10.20.22.2.5)
  3. SHALL contain exactly one [1..1] code (CONF:1198-15407)/@code="11450-4" Problem List (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-15408, CONF:1198-31141)
  4. If section/@nullFlavor is not present, SHOULD contain [0..*] entry such that it (CONF:1198-7881, CONF:1198-15505)
    1. Conforms to Problem Concern Act2 (templateId: 2.16.840.1.113883.10.20.22.4.3:2015-08-01)
  5. If section/@nullFlavor is not present, MAY contain [0..1] entry such that it (CONF:1198-30481, CONF:1198-30482)
    1. Conforms to Health Status Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.5:2014-06-09)

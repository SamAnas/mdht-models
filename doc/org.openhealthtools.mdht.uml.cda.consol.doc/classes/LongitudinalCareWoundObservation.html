Longitudinal Care Wound Observation (V2)

[Observation: templateId 2.16.840.1.113883.10.20.22.4.114]

This template represents acquired or surgical wounds and is not intended to encompass all wound types. The template applies to wounds such as pressure ulcers, surgical incisions, and deep tissue injury wounds. Information in this template may include information about the wound measurements characteristics.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-29473, CONF:1198-29474, CONF:1198-32913 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.114"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Problem Observation2 template (templateId: 2.16.840.1.113883.10.20.22.4.4:2015-08-01)
  3. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1198-31012)
  4. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1198-31013)
  5. SHALL contain exactly one [1..1] code (CONF:1198-29476)/@code="ASSERTION" assertion (CodeSystem: 2.16.840.1.113883.5.4 HL7ActCode) (CONF:1198-29477, CONF:1198-31010)
  6. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHOULD be selected from ValueSet Wound Type 2.16.840.1.113883.1.11.20.2.6 DYNAMIC (CONF:1198-29485)
  7. SHOULD contain zero or one [0..1] targetSiteCode, which SHOULD be selected from ValueSet Body Site 2.16.840.1.113883.3.88.12.3221.8.9 DYNAMIC (CONF:1198-29488)
  8. SHOULD contain [0..*] author such that it (CONF:1198-31542)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  9. SHOULD contain [0..*] entryRelationship such that it (CONF:1198-29495, CONF:1198-29496, CONF:1198-29497)
    1. Contains @typeCode="COMP" COMP
    2. Conforms to Wound Measurement Observation (templateId: 2.16.840.1.113883.10.20.22.4.133)
  10. SHOULD contain [0..*] entryRelationship such that it (CONF:1198-29503, CONF:1198-29504, CONF:1198-29505)
    1. Contains @typeCode="COMP" COMP
    2. Conforms to Wound Characteristic (templateId: 2.16.840.1.113883.10.20.22.4.134)
  11. MAY contain [0..*] entryRelationship such that it (CONF:1198-31890, CONF:1198-31891, CONF:1198-31892)

    When the wound observed is a type of pressure ulcer, then this template SHOULD contain an entry for the Number of Pressure Ulcers.

    1. Contains @typeCode="COMP" COMP
    2. Conforms to Number Of Pressure Ulcers Observation2 (templateId: 2.16.840.1.113883.10.20.22.4.76:2015-08-01)
  12. MAY contain [0..1] entryRelationship such that it (CONF:1198-31893, CONF:1198-31894, CONF:1198-31919)

    When the wound observed is a type of pressure ulcer, then this template SHOULD contain an entry for the Highest Pressure Ulcer Stage.

    1. Contains @typeCode="COMP" COMP
    2. Conforms to Highest Pressure Ulcer Stage (templateId: 2.16.840.1.113883.10.20.22.4.77)

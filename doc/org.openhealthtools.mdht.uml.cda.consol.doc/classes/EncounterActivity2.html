Encounter Activity (V3)

[Encounter: templateId 2.16.840.1.113883.10.20.22.4.49]

This clinical statement describes an interaction between a patient and clinician. Interactions may include in-person encounters, telephone conversations, and email exchanges.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-8712, CONF:1198-26353, CONF:1198-32546 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.49"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Encounter Activities template (templateId: 2.16.840.1.113883.10.20.22.4.49)
  3. SHALL contain exactly one [1..1] code, which SHOULD be selected from ValueSet EncounterTypeCode 2.16.840.1.113883.3.88.12.80.32 DYNAMIC (CONF:1198-8714)
  4. MAY contain zero or one [0..1] sDTCDischargeDispositionCode (CONF:1198-32176)

    Note: The prefix sdtc: SHALL be bound to the namespace "urn:hl7-org:sdtc". The use of the namespace provides a necessary extension to CDA R2 for the use of the dischargeDispositionCode element.

  5. SHOULD contain [0..*] participant such that it (CONF:1198-8738, CONF:1198-14903)
    1. Conforms to Service Delivery Location (templateId: 2.16.840.1.113883.10.20.22.4.32)
  6. MAY contain [0..*] entryRelationship such that it (CONF:1198-8722, CONF:1198-8723, CONF:1198-14899)
    1. Contains @typeCode="RSON" RSON
    2. Conforms to Indication2 (templateId: 2.16.840.1.113883.10.20.22.4.19:2014-06-09)
  7. MAY contain zero or more [0..*] performer such that it (CONF:1198-8725)
    1. SHALL contain exactly one [1..1] assignedEntity such that it (CONF:1198-8726)
      1. MAY contain zero or one [0..1] code, which SHOULD be selected from ValueSet Healthcare Provider Taxonomy (HIPAA) 2.16.840.1.114222.4.11.1066 DYNAMIC (CONF:1198-8727)
  8. MAY contain [0..*] entryRelationship such that it (CONF:1198-15492, CONF:1198-15973)
    1. Conforms to Encounter Diagnosis2 (templateId: 2.16.840.1.113883.10.20.22.4.80:2015-08-01)
  9. sdtc:dischargeDispositionCode SHALL contain exactly [1..1] @code, which SHALL be selected from ValueSet 2.16.840.1.113883.3.88.12.80.33 NUBC UB-04 FL17-Patient Status (code system 2.16.840.1.113883.6.301.5) DYNAMIC or, if access to NUBC is unavailable, from CodeSystem 2.16.840.1.113883.12.112 HL7 Discharge Disposition. sdtc:dischargeDispositionCode SHALL contain exactly [1..1] @codeSystem, which SHALL be either CodeSystem: NUBC 2.16.840.1.113883.6.301.5 OR CodeSystem: HL7 Discharge Disposition 2.16.840.1.113883.12.112 (CONF:1198-32177, CONF:1198-32377)

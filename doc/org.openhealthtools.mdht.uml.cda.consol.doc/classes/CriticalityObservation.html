Criticality Observation

[Observation: templateId 2.16.840.1.113883.10.20.22.4.145]

This observation represents the gravity of the potential risk for future life-threatening adverse reactions when exposed to a substance known to cause an adverse reaction in that individual. When the worst case result is assessed to have a life-threatening or organ system threatening potential, it is considered to be of high criticality.

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-32918, CONF:1198-32923 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.145"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:1198-32921)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1198-32922)
  4. SHALL contain exactly one [1..1] code (CONF:1198-32919)/@code="82606-5" Allergy or intolerance criticality (CodeSystem: 2.16.840.1.113883.6.1 LOINC) (CONF:1198-32925, CONF:1198-32926, DSTU:872)
  5. SHALL contain exactly one [1..1] statusCode (CONF:1198-32920)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1198-32927)
  6. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHALL be selected from ValueSet Criticality Observation 2.16.840.1.113883.1.11.20549 STATIC 2015-08-01 (CONF:1198-32928)

Procedure Activity Act2

[Act: templateId 2.16.840.1.113883.10.20.22.4.12]

This template represents any act that cannot be classified as an observation or procedure according to the HL7 RIM. Examples of these acts are a dressing change, teaching or feeding a patient, or providing comfort measures. The common notion of "procedure" is broader than that specified by the HL7 Version 3 Reference Information Model (RIM). Procedure templates can be represented with various RIM classes: act (e.g., dressing change), observation (e.g., EEG), procedure (e.g., splenectomy).

  1. SHALL contain exactly one [1..1] templateId ( CONF:1098-8291, CONF:1098-10519, CONF:1098-32505 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.12"
    2. SHALL contain exactly one [1..1] @extension="2014-06-09"
  2. SHALL conform to Procedure Activity Act template (templateId: 2.16.840.1.113883.10.20.22.4.12)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:1098-8290)
  4. SHALL contain exactly one [1..1] statusCode (CONF:1098-8298), which SHALL be selected from ValueSet ProcedureAct statusCode 2.16.840.1.113883.11.20.9.22 STATIC 2014-04-23 (CONF:1098-32364)
  5. SHALL contain exactly one [1..1] effectiveTime (CONF:1098-8299)
  6. MAY contain [0..1] entryRelationship such that it (CONF:1098-8322, CONF:1098-8323, CONF:1098-31396)
    1. Contains @typeCode="SUBJ" SUBJ
    2. Conforms to Instruction2 (templateId: 2.16.840.1.113883.10.20.22.4.20:2014-06-09)
  7. MAY contain [0..*] entryRelationship such that it (CONF:1098-8326, CONF:1098-8327, CONF:1098-15601)
    1. Contains @typeCode="RSON" RSON
    2. Conforms to Indication2 (templateId: 2.16.840.1.113883.10.20.22.4.19:2014-06-09)
  8. MAY contain [0..*] entryRelationship such that it (CONF:1098-8329, CONF:1098-8330, CONF:1098-15602)
    1. Contains @typeCode="COMP" COMP
    2. Conforms to Medication Activity2 (templateId: 2.16.840.1.113883.10.20.22.4.16:2014-06-09)
  9. SHOULD contain zero or more [0..*] performer such that it (CONF:1098-8301)
    1. SHALL contain exactly one [1..1] assignedEntity such that it (CONF:1098-8302)
      1. SHALL contain at least one [1..*] id (CONF:8303, R2.0=CONF:1098-8303)
      2. SHALL contain exactly one [1..1] addr (CONF:8304, R2.0=CONF:1098-8304)
      3. SHALL contain at least one [1..*] telecom (CONF:1098-8305)
      4. SHOULD contain zero or one [0..1] representedOrganization such that it (CONF:1098-8306)
        1. SHOULD contain zero or more [0..*] id (CONF:8307, R2.0=CONF:1098-8307)
        2. MAY contain zero or more [0..*] name (CONF:8308, R2.0=CONF:1098-8308)
        3. SHALL contain at least one [1..*] telecom (CONF:1098-8310)
        4. SHALL contain at least one [1..*] addr (CONF:1098-8309)
  10. SHOULD contain [1..*] author such that it (CONF:1098-32477)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  11. entryRelationship with target class Instruction2 SHALL contain exactly one [1..1] @inversionInd="true" true (CONF:1098-8324)
  12. @code MAY be selected from CPT-4 (CodeSystem: 2.16.840.1.113883.6.12) or ICD10 PCS (CodeSystem: 2.16.840.1.113883.6.4) or CDT-2 (Code System: 2.16.840.1.113883.6.13) (CONF:1098-19190)

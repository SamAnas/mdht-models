Cognitive Abilities Observation

[Observation: templateId 2.16.840.1.113883.10.20.22.4.126]

The Cognitive Abilities Observation represents a patient's ability to perform specific cognitive tasks (e.g., ability to plan, logical sequencing ability, ability to think abstractly).

REMOVED

  1. SHALL contain exactly one [1..1] templateId ( CONF:29248, CONF:29249 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.126"
  2. SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: 2.16.840.1.113883.5.6 HL7ActClass) (CONF:29246)
  3. SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: 2.16.840.1.113883.5.1001 HL7ActMood) (CONF:29247)
  4. SHALL contain at least one [1..*] id (CONF:29250)
  5. SHALL contain exactly one [1..1] code, which SHOULD be selected from ValueSet Cognitive Abilities Value Set 2.16.840.1.113883.11.20.9.48 DYNAMIC (CONF:29251)
  6. SHOULD contain zero or one [0..1] text (CONF:29252)
  7. SHALL contain exactly one [1..1] statusCode (CONF:29256)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:29257)
  8. SHALL contain exactly one [1..1] effectiveTime (CONF:29258)
  9. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHOULD be selected from ValueSet Mental and Functional Status Response 2.16.840.1.113883.11.20.9.44 DYNAMIC (CONF:29264)
  10. SHOULD contain [0..*] author such that it (CONF:31923)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  11. MAY contain [0..*] entryRelationship such that it (CONF:29266, CONF:29267, CONF:29268)
    1. Contains @typeCode="COMP" COMP
    2. Conforms to Assessment Scale Observation (templateId: 2.16.840.1.113883.10.20.22.4.69)

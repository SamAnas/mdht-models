Problem Observation (V3)

[Observation: templateId 2.16.840.1.113883.10.20.22.4.4]

This template reflects a discrete observation about a patient's problem. Because it is a discrete observation, it will have a statusCode of "completed". The effectiveTime, also referred to as the "biologically relevant time" is the time at which the observation holds for the patient. For a provider seeing a patient in the clinic today, observing a history of heart attack that occurred five years ago, the effectiveTime is five years ago. The effectiveTime of the Problem Observation is the definitive indication of whether or not the underlying condition is resolved. If the problem is known to be resolved, then an effectiveTime/high would be present. If the date of resolution is not known, then effectiveTime/high will be present with a nullFlavor of "UNK".

  1. SHALL contain exactly one [1..1] templateId ( CONF:1198-14926, CONF:1198-14927, CONF:1198-32508 ) such that it
    1. SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.4"
    2. SHALL contain exactly one [1..1] @extension="2015-08-01"
  2. SHALL conform to Problem Observation template (templateId: 2.16.840.1.113883.10.20.22.4.4)
  3. MAY contain zero or one [0..1] @negationInd (CONF:1198-10139)

    The negationInd is used to indicate the absence of the condition in observation/value. A negationInd of "true" coupled with an observation/value of SNOMED code 64572001 "Disease (disorder)" indicates that the patient has no known conditions.

  4. SHALL contain exactly one [1..1] effectiveTime (CONF:1198-9050)

    If the problem is known to be resolved, but the date of resolution is not known, then the high element SHALL be present, and the nullFlavor attribute SHALL be set to 'UNK'. Therefore, the existence of an high element within a problem does indicate that the problem has been resolved.

  5. SHALL contain exactly one [1..1] value with @xsi:type="CD", which SHOULD be selected from ValueSet Problem 2.16.840.1.113883.3.88.12.3221.7.4 DYNAMIC (CONF:1198-9058)
  6. SHALL contain exactly one [1..1] code, which SHOULD be selected from ValueSet Problem Type (SNOMED CT) 2.16.840.1.113883.3.88.12.3221.7.2 STATIC 2012-06-01 (CONF:1198-9045)
  7. contain zero or one [0..1] text
  8. SHALL contain exactly one [1..1] statusCode (CONF:1198-9049)/@code="completed" Completed (CodeSystem: 2.16.840.1.113883.5.14 ActStatus) (CONF:1198-19112)
  9. SHOULD contain [0..*] author such that it (CONF:1198-31147)
    1. Conforms to Author Participation (templateId: 2.16.840.1.113883.10.20.22.4.119)
  10. MAY contain [0..1] entryRelationship such that it (CONF:1198-29951, CONF:1198-31531, CONF:1198-29952)
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Prognosis Observation (templateId: 2.16.840.1.113883.10.20.22.4.113)
  11. MAY contain [0..*] entryRelationship such that it (CONF:1198-31063, CONF:1198-31532, CONF:1198-31064)
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Priority Preference (templateId: 2.16.840.1.113883.10.20.22.4.143)
  12. MAY contain [0..1] entryRelationship such that it (CONF:1198-9063, CONF:1198-9068, CONF:1198-15591)
    1. Contains @typeCode="REFR" REFR
    2. Conforms to Problem Status (templateId: 2.16.840.1.113883.10.20.22.4.6)
  13. contain [0..1] entryRelationship such that it
    1. Conforms to Health Status Observation (templateId: 2.16.840.1.113883.10.20.22.4.5)
  14. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
  15. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
  16. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
  17. effectiveTime, if present, SHALL contain exactly one [1..1] low (CONF:1198-15603)
  18. effectiveTime, if present, MAY contain zero or one [0..1] high (CONF:1198-15604)
  19. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
  20. MAY satisfy: This constraint is being overridden as it has been removed in v2 and should never fire
